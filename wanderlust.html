Wanderlust

Wanderlust is an alternative navigation mobile application that encourages serendipity while ensuring that you get to your destination on time. Project Website

Timeframe: 10 weeks

What did I do? - I worked in a team of 5 developers and designers to conduct design research, design thhe user experience and interface, conduct usability tests and code the frontend of the React Native mobile application.

Introduction

I joined the project after it had gone through an initial 10-week design cycle. The summary:

Problem
  • The fastest journey is not always the most fun or the safest
  • Using traditional navigation applications doesnt allow serendipity or immersion
Features
  • Directions without set route
  • Draw boundary to restrict wandering path
  • Filters shows areas clustered together to indicate safety,attractions, residencies etc
  • Push notifications to warn user of breaching boundary
