Lightup

Lightup is an ed tech startup that makes educational toys and games for children. Timeframe: 6 months

For how long - I worked as a game design and development intern for a new toy called Hololab which is an Augmented reality-based game that uses tangible objects to teach children different topics revolving around science.

What did I use - Unity game engine

I worked on designing and prototyping (in Unity) a level that teaches the concepts of the conservation of energy. The players have to use various energy sources (wind, nuclear,solar etc)from an island to power a magnetic field generator to save the earth's dying magnetic field

I started of with using the Next Generation Science standards to define the learning objectives of the game
